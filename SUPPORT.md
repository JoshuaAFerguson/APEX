# Support

## How to get help

### Questions and Discussions

If you have questions about using APEX or want to discuss features, please:

1. Check the [Documentation](docs/)
2. Search existing [Issues](https://github.com/JoshuaAFerguson/apex/issues) to see if someone else has had the same question.
3. Open a new Issue with the "question" or "help wanted" label.

### Reporting Bugs

If you believe you have found a bug:

1. Search existing [Issues](https://github.com/JoshuaAFerguson/apex/issues) to see if it has already been reported.
2. If not, open a new Issue using the Bug Report template.
3. Provide as much information as possible (version, reproduction steps, logs).

### Feature Requests

We welcome feature requests! Please open an Issue with the "enhancement" label to propose new features.
